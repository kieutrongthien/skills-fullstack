# Skills - Full-Stack

This repository contains a comprehensive collection of skills and knowledge required for full-stack development. It covers both front-end and back-end technologies, frameworks, and best practices.

## Skills Included

- **Agile Planning**: Create epics, user stories, and estimate points for predictable delivery. [Learn more](./agile-planning/SKILL.md)
- **Agile Development Execution**: Guide coding execution, track story progress, and keep work flowing. [Learn more](./agile-development-execution/SKILL.md)
- **Sprint Management**: Plan, run, and close sprints with healthy velocity and burndown. [Learn more](./sprint-management/SKILL.md)
- **Progress Reporting**: Compile and communicate project status, metrics, and risks to stakeholders. [Learn more](./progress-reporting/SKILL.md)
- **Task Completion Verification**: Verify acceptance criteria, run tests, and ensure quality before marking tasks as done. [Learn more](./task-completion-verification/SKILL.md)
- **Front-End Development**: Master HTML, CSS, JavaScript, and popular frameworks like React and Vue.
- **Back-End Development**: Work with server-side languages, databases, and APIs using NestJS, Laravel, Golang and more.
